Maths Programme

The Mathematics Support Programme makes math fun for children. Simple concepts like numbers, place value, mathematical functions are taught using an activity based method with tools like a counting board and number grids, which help children comprehend mathematics.

Library Programme

The School-based Library Programme is a key part of Akshara's approach, as we believe that books and reading impact children's learning outcomes by helping them become better readers. The Programme has set up nearly 400 libraries in the 1417 Government Primary Schools in Bangalore Urban district covering a total of 215,000 children. Every child in every government primary school (Classes 2-7 ) in Bangalore has access to library facilities.
